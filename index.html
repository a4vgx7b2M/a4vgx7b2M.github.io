ESP Web Flasher

Simple site for flashing of Gaggiuino software any ESP32 not purchased through an approved supplier
See gaggiuino-esp32-loader for more details
Requires an ESP32-8048S043 screen for the full version, or requires any ESP32-S3-N16R8 module for the headless version
Do not use this site on screens/headless modules purchased from an approved supplier of you will brick your ESP32

Please be aware that this site is not affiliated with or endorsed by the official Gaggiuino project in any way

Install

Connect your ESP32, select the correct device, and hit install

Installer powered by ESP Web Tools.
